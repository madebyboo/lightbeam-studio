Order Flow

  1. Size (12oz / 16oz)
  2. Hot or Iced
  3. Base (Matcha / Coffee)
  4. Milk
  5. Sweetener (optional)
  6. Flavor (optional)
  7. Topping (optional)
  8. Repeat back order before making

Build Standards

Matcha 12oz2g + 1oz water
Matcha 16oz3g + 1.5oz water
Coffee 12oz2 shots
Coffee 16oz3 shots
Sweetness 12oz0.5 oz
Sweetness 16oz0.75 oz
Cream Top0.75 / 1.0 oz

Build Sequence

Iced Matcha

  1. Whisk matcha + hot water
  2. Add sweetener/flavor
  3. Add ice
  4. Add milk
  5. Pour matcha
  6. Add topping + garnish

Iced Coffee

  1. Pull espresso
  2. Add sweetener/flavor
  3. Add ice + milk
  4. Pour espresso
  5. Add topping

QC Checklist

Correct name Correct size/temp Balance check Clean handoff Photo-ready look
